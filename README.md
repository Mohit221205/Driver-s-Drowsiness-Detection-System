# Driver-s-Drowsiness-Detection-System
Simple Python project to detect whether the Driver is consious or sleepy. If sleepy then the beep sound will generated till he/she becomes consious.

Simple Hardware components like Camera And Speaker are used.
Project include the use of Bulit-in Libraries like opencv, winsound.

OpenCV is used to Capture, Process the Image.
WinSound is used for the Beep sound.
